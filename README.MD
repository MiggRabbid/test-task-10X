#### Задача
Сверстать и оживить небольшой блок точно по макету в Figma: https://www.figma.com/design/iuXLXifXgv4ZCzad0KeYKr/Test-Task?node-id=0-1&p=f&t=KsNXMs8WQDm1CVOJ-0

Это типичный карточный каталог с поиском и фильтрами.

Основной стек:
- HTML
- JavaScript (vanilla)
- SCSS
Без библиотек и фреймворков.

#### Требования к функциональности
Pixel-perfect вёрстка по макету (1920 px → до мобильных 320 px) Адаптив «резиновый» (без жёстких брейкпоинтов, плавно масштабируется) Поиск по названию карточек (живой, без перезагрузки) Фильтрация по категориям (всё сгруппировано по категориям) Чистый и понятный код по https://ru.bem.info/methodology/

#### Реализация
1. HTML с базовой вёрсткой
2. Отдельные скрипты для стейта, фильтров и карточек.
3. Стилизация в SCSS
